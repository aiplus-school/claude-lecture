## [P2] 今日のお題 — アジェンダのMDファイルを渡すと会議用HTMLサイトが完成する

今日やることを一言で言うと、「会議のアジェンダをMarkdownファイルにしてClaude Code に渡し、資料をHTMLサイトに変換すること」です。

Markdownファイルは人間にとってもAIにとっても読みやすい方式のファイルのことです。まず事前に会議のアジェンダをClaudeと壁打ちしてMarkdownファイルにしておきます。このアジェンダのMarkdownファイルをClaude Code に渡して「HTMLの資料を作成して」と依頼することで、ブラウザで開ける会議資料に仕上がります。

今日は、架空のディスカウントストア「驚安堂」の2026年夏・新商品リリース説明会議のアジェンダを事前に用意しました。このファイルを使って実演します。

## [P4] クロージング — 設定より、まず手を動かそう

はじめて Claude Code を使ってみて、どうでしたか？

細かいことは、あとから覚えれば十分です。

Claude Code の使い方を覚えるのに一番いい方法は、使いながら覚えることです。分からないことが出てきたら、そのまま Claude Code に聞きましょう。詰まったら「こういうエラーが出ました」と貼り付けて聞くことで、Claude Code で成果物を作ることにどんどん慣れてきます。
