# CLAUDE.md — AI講座 動画制作プロジェクト

このリポジトリは「Claude Code 追加講座」の動画制作プロジェクト。
本ファイルは共通の作業ガイドライン（WBS・制作方針・参考資料）をまとめたもの。

## プロジェクト概要

Claude Code と各ツールを組み合わせた動画を **計3本** 制作する。
各テーマ1本。制作順は下記フォルダの並び順どおり。

| 順番 | テーマ | フォルダ |
|---|---|---|
| 1 | n8n MCP × Claude Code | `n8n MCP x Claude Code/` |
| 2 | Remotion × Claude Code | `Remotion x CC/` |
| 3 | HiggsField MCP × Claude Code | `HiggsField MCP x CC/` |

工数前提：テーマごとに「リサーチ・内容決定 3〜4h ＋ 制作 5〜6h = 8〜10h」（上司見積もりベース）。
全テーマ合計：24〜30h。

## WBS（各テーマ共通の作業フロー）

各テーマは以下6タスク・2フェーズで構成される（担当：永藤）。

### フェーズ1. リサーチ・内容決定（3.5h）
| No | タスク | 成果物 | 工数(下限〜上限) |
|---|---|---|---|
| x-1 | 概要・実用事例のリサーチ | リサーチメモ | 1〜1h |
| x-2 | 実際に使ってみる（検証） | 検証ログ・サンプル | 2〜2.5h |
| x-3 | 講座で扱う内容の決定 | 内容アウトライン | 0.5〜0.5h |

### フェーズ2. 制作（台本・収録・スライド）（4.5〜6h）
| No | タスク | 成果物 | 工数(下限〜上限) |
|---|---|---|---|
| x-4 | 台本作成 | 台本 | 2〜2.5h |
| x-5 | スライド作成 | スライド | 1.5〜2h |
| x-6 | 画面収録 | 収録データ | 1〜1.5h |

（`x` はテーマ番号 1=n8n / 2=Remotion / 3=HiggsField）

各テーマフォルダには、上記成果物（リサーチメモ／検証ログ／アウトライン／台本／スライド／収録データ）を格納していく。

## 動画制作の方針・注意点

- **手順書にしない。** 「なぜこのボタンを選ぶのか」など簡単な理由説明を入れる。
- **専門用語はなるべく避ける。** 使う場合は注釈を入れる、または概要欄に用語と説明を記載する。
- **馴染みの薄いアプリ**は、概要欄に注釈やリンクを貼る。
- **改善前提の注意点**を最後に説明する（受講者が自分で試しながら改善していくことを前提に）。
- **主ツール以外を使う理由**を説明する。「なぜそれを使うのか」「主ツールでなくそのツールを使うメリット」を伝えると受講者が納得しやすい。
