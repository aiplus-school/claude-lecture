## テーマ別の詳細

### 1. n8n MCP × Claude Code

| 方法 | 月額 | APIキー | 自動デプロイ | 難易度 |
|---|---|---|---|---|
| **セルフホスト（自PC / Docker）** | **0円** | ✅ 自分で発行 | ✅ | 高 |
| セルフホスト（VPS） | $5〜7（約800〜1,100円） | ✅ 自分で発行 | ✅ | 高 |
| **n8n Cloud Starter** | **€20**（年払）／€24（月払） | ✅ | ✅ | **低** |
| n8n Cloud 無料トライアル（14日） | 0円 | ❌ | ❌ 手動コピペ | 低 |
| **MCPのみ（n8n本体なし）** | **0円** | — | — | 最低 |

**⚠️ 訂正：「APIキーがないとMCPが使えない＝課金必須」は誤りだった。**

| | APIキー**なし** | APIキー**あり** |
|---|---|---|
| 使えるツール | **コア7個**（ノード検索、`validate_workflow` 等） | 上記 + 管理16個 |
| できること | **設計・構築・検証（design, build, validate）** | 左記 + **自動デプロイ** |
| n8nへの入れ方 | JSONを**UIに手動で貼り付け** | **自動反映** |

→ **APIキーの有無で変わるのは「最後の一手間」だけ。** MCPの本質（実在の仕様を参照する、検証する）はAPIキーなしで全部使える。
→ **さらにセルフホストなら自分でAPIキーを無料発行できる。** よって **課金は必須ではない。**

- Community Edition（セルフホスト）は**完全無料・実行無制限・500以上の連携も全部入り**。
- 実行回数は「ワークフロー1起動＝1実行」。**学習用途で Starter の2,500回/月を使い切ることはまずない。**
- 案B（AI分類）を採用するなら OpenAI API が別途従量課金（学習用途なら月数十〜数百円）。ただし **Cloud Starterには AIクレジットが月2,300分含まれる**（これで足りるか要検証）。

> "A Seat covers one person who writes Remotion code themselves or **uses agentic coding tools**."

**本講座の受講者がまさにこの定義に該当する。** 4人以上の会社で業務利用するなら有料。

**追加クレジット：100クレジット ≈ $5（有効期限90日）→ 1クレジット ≈ 約8円**
**未使用クレジットは翌月に繰り越されない。解約時は没収。**

**この動画の内容（画像5枚 + 動画1本 + 素材2〜3枚）の試算**
| | 安く済ませた場合 | 高くついた場合 |
|---|---|---|
| 1回で成功 | 約6〜9クレジット | 約50〜110クレジット |
| **試行錯誤3倍で見る** | **約20〜30クレジット** | **約150〜330クレジット** |

✅ **調査で判明（推奨プランが Starter → Plus に変更）**
- **クレジットの"量"だけなら Starter（200）で足りる**（試算は20〜30クレジット）
- **Starter を選べない理由は"量"ではなく"モデル制限"**

### Starter のモデル制限（確度別）

| | モデル | 確度 |
|---|---|---|
| ❌ **使えない** | **Veo 3 / Veo 3 Fast / Veo 3.1、Sora 2、Seedance 2.0** | **高**（公式ブログがPlusの目玉として列挙、vo3ai「No Veo 3」） |
| ⭕ 使える | **Kling 3.0**（月約23本の5秒動画）、**Nano Banana Pro**（月約100枚） | 中（vo3ai・costbenchが一致、公式明記なし） |
| 🚨 **不明** | **Soul** ← **最重要** | vo3ai・kujaku-x・公式ブログが揃って「**Plusから**」を示唆 |

**各社の表現**：Starter＝「core generation models」「selected models only」「6+ included」／ Plus＝「**all models on the platform**」

🚨 **Soul が Plus 限定なら、費用試算の前提が崩れる。**
「Soulは1クレジットで4枚と激安だからStarterで安く済む」と整理していたが、**そのSoul自体がPlus限定なら、Starterで安く済ませる作戦は成立しない。**
→ **収録前に最優先で実物確認。**

⚠️ **正確なモデル一覧を公表している第三者ソースは存在しない。** 各社「selected models」としか書いていない。**公式ページはJS描画で読めず、ログイン状態での目視確認が唯一の確実な方法。**

✅ **無料枠について**：**正式なFreeプランは料金ページに存在しない**（imagine.art・公式ブログ）。新規ユーザーに少量の日次クレジットが付くのみで、**透かしあり＋商用利用不可**。→ **「無料で試せます」とは言えない。**

🚨 **最大の落とし穴：AIに「20パターン作って」と頼むと、その分だけ実費が飛ぶ。**

## 🚨 制作方針：動画本編に金額を入れない

**調査の結論として、料金情報は驚くほど不安定。**

- **Higgsfield は2026年1〜4月にプラン名を全面改称**（`Basic/Pro/Ultimate/Creator` → `Starter/Plus/Ultra/Business`）
- Scopeful曰く **「Higgsfieldは2026年だけで"Unlimited"の定義を最低3回変えている」**
- 第三者情報がエントリープランだけで **$5 / $9 / $15 / $19** と全く一致しない
- 期間限定オファーが常時走っている

→ **動画本編に具体的な金額を入れると、公開直後に陳腐化して信頼を落とす。**
→ **金額は概要欄に書く**（後から直せる）。**動画内では課金の"構造"だけを説明する。**

| テーマ | 動画内で言うこと | 概要欄に書くこと |
|---|---|---|
| n8n | 「セルフホストなら無料。手軽に始めたいなら有料プラン」 | 具体的なプラン名と金額 |
| Remotion | 「個人は無料。**会社の人数で有料になる**」 | シート定義と金額 |
| HiggsField | 「**クレジット制で、生成のたびに実費がかかる**」 | プラン名と金額、クレジット消費の目安 |
